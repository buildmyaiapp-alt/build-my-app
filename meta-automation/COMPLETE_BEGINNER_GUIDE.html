Create Your Facebook Page & Business Account

This is the foundation — your "business identity" on Meta

1

Create a Facebook Page for your workshop

You need a Facebook Page (not a personal profile) to run ads.

  1. Go to facebook.com and log in with your personal account
  2. Click the ➕ (Create) button in the top right
  3. Click "Page"
  4. Page Name: Dream App Workshop
  5. Category: type "Education" → select it
  6. Click Create Page
  7. Add a profile photo (your photo or logo)
  8. Add a cover photo (use your landing page banner image)
Page created! Now copy your Page ID — go to your Page → About → scroll down → you'll see "Page ID: 123456789"
2

Create Meta Business Manager

Business Manager is the control center for all your ads.

https://business.facebook.com
  1. Click "Create Account"
  2. Business Name: Dream App Workshop
  3. Your Name & Email → Click Submit
  4. Verify your email when Facebook sends a confirmation
  5. After login → Go to Business Settings (gear icon)
  6. Click Pages → Add → Add a Page → select the page you just created
Use the same Gmail account you use daily — you'll get notifications here
3

Create an Ad Account

  1. In Business Settings → click Ad Accounts
  2. Click + Add → Create a new ad account
  3. Name: Dream App Workshop Ads
  4. Time zone: India Standard Time
  5. Currency: Indian Rupee (INR)
  6. Click Create
  7. Copy your Ad Account ID — it looks like act_123456789
You must add a payment method before running ads. Go to Ad Account → Payment Settings → Add Credit/Debit Card or UPI
4

Add Payment Method

  1. In Business Settings → Payments
  2. Click Add Payment Method
  3. Choose: Credit/Debit Card (Visa/Mastercard works best) or PayPal
  4. Enter card details → Save
  5. Meta will charge a small verification amount (Rs 1–2) that gets refunded
Meta India also accepts UPI for some accounts. If you see the UPI option, use that.

Get Your Meta Access Token (API Key)

This allows our automation script to control your ads automatically

5

Create a Meta Developer App

https://developers.facebook.com
  1. Log in with the same Facebook account
  2. Click My Apps → Create App
  3. Select: "Business" → Next
  4. App Name: DreamAppAdsAutomation
  5. App Contact Email: your email
  6. Business Account: select your Business Manager
  7. Click Create App
You now have a Developer App. Stay on this page.
6

Add Marketing API to your App

  1. Inside your app, find the left menu → click "Add Product"
  2. Find "Marketing API" → click Set Up
  3. This adds the ability to create and manage ads via code
7

Generate Your Access Token

  1. In the left menu → Tools → Graph API Explorer
  2. Top right dropdown: select your App (DreamAppAdsAutomation)
  3. Click "Generate Access Token"
  4. A popup appears — check ALL these permissions:
    • ✅ ads_management
    • ✅ ads_read
    • ✅ leads_retrieval
    • ✅ pages_manage_ads
    • ✅ pages_read_engagement
  5. Click Generate Token → Copy the token (very long string)
This token expires in 1 hour. You need to convert it to a Long-Lived Token (60 days). Do the next step immediately!
8

Convert to Long-Lived Token (60 days)

Copy this URL and fill in your details, then open it in your browser:

https://graph.facebook.com/v19.0/oauth/access_token ?grant_type=fb_exchange_token &client_id=YOUR_APP_ID &client_secret=YOUR_APP_SECRET &fb_exchange_token=YOUR_SHORT_TOKEN

Where to find these:

  • App ID: In your Developer App → Settings → Basic → App ID
  • App Secret: Same page → App Secret → click "Show"
  • Short Token: The token you just generated in Step 7

Open the URL in browser → you'll get a JSON response with "access_token" → copy that long token

This is your permanent Long-Lived Token. Save it safely — paste into config.py
3

Host Your Landing Page Online

Your landing page needs a real URL — not just a file on your computer

9

Host on Netlify (Free — takes 5 minutes)

https://netlify.com
  1. Sign up with your Gmail
  2. On the dashboard → find "Deploy manually" section
  3. Drag and drop your gym app folder (the one with index.html)
  4. Netlify automatically deploys it and gives you a URL like:
    https://dreamapp-workshop.netlify.app
  5. Copy that URL → paste into LANDING_PAGE_URL in config.py
You can get a custom domain later (Rs 700/year from GoDaddy) like dreamappworkshop.in
10

Add Meta Pixel to your Landing Page

Pixel tracks who visits your page — essential for retargeting and optimization.

  1. In Business Manager → Events Manager
  2. Click + Connect Data Source → Web
  3. Select Meta Pixel → name it: DreamApp Pixel
  4. Enter your website URL → Click Continue
  5. Choose: Manually add Pixel code to website
  6. Copy the Pixel code (looks like a script tag)
  7. In your index.html, paste it just before </head>
  8. Redeploy to Netlify (drag and drop again)
Now Meta can track conversions and optimize your ads automatically!

Get Anthropic API Key (for Claude AI)

Claude AI will write your ad copies automatically

11

Create Anthropic Account & Get API Key

https://console.anthropic.com
  1. Sign up with your Gmail
  2. Verify email
  3. Go to API Keys in the left menu
  4. Click Create Key
  5. Name: MetaAdsAutomation
  6. Copy the key (starts with sk-ant-...)
  7. Paste into ANTHROPIC_API_KEY in config.py
You get $5 free credits to start. For regular use, add a card — Claude API costs ~Rs 0.50 per ad copy set generated.

Set Up WhatsApp Auto-Reply (WATI)

Every new lead gets an automatic WhatsApp message within minutes

12

Sign up for WATI.io

https://wati.io
  1. Click Start Free Trial (14 days free)
  2. Sign up with Gmail
  3. Connect your WhatsApp Business number:
    • You need a number NOT already registered on regular WhatsApp

Cheapest plan is Rs 999/month. You can also use Interakt.co (similar pricing) or AiSensy.com

https://adsmanager.facebook.com

  4. Form Type: More Volume (gets more leads)

Install dependencies & Run

Open Terminal (Mac) → navigate to the folder:

cd Desktop/gym\ app/meta-automation pip install -r requirements.txt python run.py

You'll see this menu:

1️⃣ Generate AI ad copies (Claude) 2️⃣ Launch new campaign 3️⃣ Activate paused campaign (go LIVE) 4️⃣ Check today's performance 5️⃣ Process new leads + WhatsApp 6️⃣ View leads summary 7️⃣ Start auto-scheduler
18

Generate AI Ad Copies — Press 1

  1. Press 1 → Enter
  2. Type 5 → Enter (5 ad variants)
  3. Claude AI generates 5 different ad copy versions in ~30 seconds
  4. You'll see them printed — review them
  5. They're automatically saved to generated_copies.json
Claude writes different angles: fear, curiosity, proof, dream outcome, price shock
19

Launch Campaign — Press 2

  1. Press 2 → Enter
  2. Image path: press Enter to skip (or type path to your image file)
  3. Go live immediately? Type N → Enter (start paused)
  4. Script creates your campaign on Meta automatically
  5. Campaign ID is saved to campaign_ids.json
Always start PAUSED first. Go to Ads Manager and REVIEW the campaign before going live. Check targeting, copy, budget.
20

Review in Ads Manager & Go Live — Press 3

  1. Go to adsmanager.facebook.com
  2. Find your campaign: "Dream App Workshop – [date]"
  3. Click into it → review the ad sets and ads
  4. Check: targeting, daily budget, ad copy, link
  5. If everything looks good → come back to terminal
  6. Press 3 → type y → Enter
  7. 🎉 Your ads are now LIVE!
Ads go live! Meta starts showing them to your target audience immediately. First results appear in 1–3 hours.

What Images/Videos to Use in Your Ads

Creative is the most important factor in ad performance

24

Best Performing Ad Formats for Your Workshop

Option A — Your Face (Best for trust) Recommended

  • Record a 15–30 second video of yourself talking to camera
  • Script: "Kya aap bina coding ke apna app banana chahte ho? Main aapko 30 minutes mein sikhaunga..."
  • Shoot in good light (near a window), simple background
  • Add captions (use CapCut app — free)

Option B — Screen Recording Easy

  • Record your screen while building an app using AI
  • Show: type prompt → AI generates app → done in minutes
  • Add voiceover: "Dekho maine sirf yeh type kiya aur app ban gaya!"

Option C — Before/After Image

  • Left side: "Developer = Rs 2 Lakh + 3 months"
  • Right side: "Our workshop = Rs 249 + 30 minutes"
  • Create free on Canva.com
Videos get 3x more leads than images. Even a simple 30-second phone video of YOU talking works better than a designed image.
